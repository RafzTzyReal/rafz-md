# RAFZ MD - Professional Downloader & Uploader Telegram Bot

Aplikasi robot Telegram multi-fungsional premium untuk mendownload media TikTok bebas watermark, serta melakukan *upload* gambar/video ke cloud storage publik melalui server `rafzuploader.lovable.app`.

## 🚀 Fitur Utama
1. **Force Join Group**: Penguncian sistem fitur otomatis sebelum user terdaftar sebagai member grup tujuan.
2. **TikTok Downloader Engine**: Mengunduh video berkualitas tinggi langsung dalam format MP4.
3. **Uploader Cloud System**: Mengubah file internal foto/video Telegram menjadi link static eksternal.
4. **Anti Spam Proteksi**: Pembatasan request rate limit untuk meredam kebanjiran trafik (DDOS internal).
